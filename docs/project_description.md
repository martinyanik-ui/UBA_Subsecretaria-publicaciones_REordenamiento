# Documentación del Proyecto

## Objetivo del Sistema
Implementar un sistema de análisis de datos para optimizar la producción en el Taller de Imprenta de la UBA.

## Problema a Resolver
- Falta de visibilidad del estado de producción
- Gestión manual de stock de materiales
- Reportes operativos no automatizados
- Dificultad para análisis de tendencias históricas

## Solución Propuesta
1. **Digitalización** de datos de producción
2. **Automatización** de reportes y dashboards
3. **Análisis predictivo** para planificación
4. **Sistema centralizado** de información

## Usuarios Beneficiados
- Personal del taller de imprenta
- Dirección de publicaciones
- Departamentos académicos
- Administración de la facultad
